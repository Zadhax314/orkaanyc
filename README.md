# Orkaanyc
**A constructed language focused on precision over amiguity.**

## Introduction
Orkaanyc is definetly a work-in-progress. I started this project in Fall of 2025, originally as a language to write my diary in. The language prioritises precision over ambiguity; I'd often take a look at things I don't like in English and change them here. The language is meant to have a sort of Nothern European sound to it, and it was partially inspired by videos of people speaking Old English. I also eventually decided to write lore around the fictional speakers of the language, which I will include in this repository.
